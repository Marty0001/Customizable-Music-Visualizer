<br>

<h3>Bars can shoot sparks when they grow. Experiment with different spark behavior combinations for unique effects, or set them to random!</h3>

<span>Spark properties:</span>
<ul>
    <li>Size</li>
    <li>Spawn rate</li>
    <li>Limit</li>
    <li>Gravity</li>
    <li>Velocity</li>
    <li>Fade rate</li>
    <li>Swade motion</li>
</ul>
